# Todo application using Java

Exposes below api endpoints

__Get All__<br />
`http://localhost:8080/todo`

__Create__<br />
`http://localhost:8080/todo`
```
{
	"title": "Finish writing",
	"done": false
}
```

__Update__<br />
`http://localhost:8080/todo`
```
{
	"id": 1,
        "title": "Finish writing",
        "done": false
}
```

__Delete__<br />
`http://localhost:8080/todo/1`
